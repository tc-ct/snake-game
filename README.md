# Classic Snake

一个可直接部署分享的经典贪吃蛇小游戏。

## 本地运行

需要先安装 Node.js。

```powershell
cd D:\AI_project
node server.js
```

浏览器打开：

`http://localhost:3000`

## 本地测试

```powershell
node --test
```

## 手动检查清单

- 方向键或 `WASD` 可以控制移动
- 吃到食物后分数会增加，蛇会变长
- 撞墙或撞到自己会结束游戏
- 按 `Space` 可以暂停或继续
- 点击 `Restart` 可以重新开始
- 手机上可以点击屏幕按钮控制方向
